我将在这里留下脚本文件备份以及精简的教程。

v2ray一键脚本（2333版本）：
bash <(curl -s -L https://git.io/v2ray.sh)
v2ray一键脚本（wulabing版本）：
wget -N --no-check-certificate -q -O install.sh "https://raw.githubusercontent.com/wulabing/V2Ray_ws-tls_bash_onekey/master/install.sh" && chmod +x install.sh && bash install.sh
ss一键脚本：
bash <(curl -s -L https://git.io/ss.sh)
SSR一键脚本：
wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/ssr.sh && chmod +x ssr.sh && bash ssr.sh
BBR多合一脚本：
wget -N --no-check-certificate "https://raw.githubusercontent.com/chiakge/Linux-NetSpeed/master/tcp.sh"
chmod +x tcp.sh
./tcp.sh

(更新的bbr新脚本可以看我另一篇专门的bbr加速博客）

针对centos7更换内核以使用BBR
rpm -ivh http://xz.wn789.com/CentOSkernel/kernel-3.10.0-229.1.2.el7.x86_64.rpm --force
rpm -qa | grep kernel #查看内核是否安装成功
reboot #重启
